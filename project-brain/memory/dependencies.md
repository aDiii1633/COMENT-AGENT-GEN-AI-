# System Dependencies

## Frontend Packages
* `react` & `react-dom` (^18.x)
* `react-router-dom` (^6.x)
* `tailwindcss` (3.4.15) - PostCSS plugin integration locked to prevent version 4 compiler conflicts.
* `lucide-react` - Unified icon system.
* `@insforge/sdk` - Backend database client.

## Backend Packages
* `fastapi` - API framework.
* `uvicorn` - ASGI server.
* `pydantic` - Schemas and environment configurations.
* `google-adk` - Agent definitions.
* `google-generativeai` - Gemini 2.5 API connector.
