# COMET Project Overview

## Core Product Identity
COMET is a Multi-Agent Business Orchestrator designed to consolidate startup planning, market research, development setup, content strategy, and pitch creation into a single unified workspace.

## Core Target Users
* **Startup Founders**: Seek dynamic GTM roadmaps and pricing models.
* **Freelancers**: Seek target demographic scoping and outlines.
* **Students**: Seek Lean Canvas layouts and structured specifications.
* **Product Teams**: Seek automated database design and Mermaid charts.

## Core MVP Agents
1. **Research Agent**: Competitor analysis, TAM/SAM/SOM.
2. **Strategy Agent**: Lean Canvas, roadmap, pricing tiers.
3. **Content Agent**: Taglines, social calendars, email copies.
4. **Development Agent**: Database schemas, system design charts.
5. **Pitch Agent**: Pitch decks, financial projections, elevator pitches.
