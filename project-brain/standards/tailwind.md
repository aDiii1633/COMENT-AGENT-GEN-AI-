# Tailwind CSS Standards

* Lock configuration to **Tailwind v3.x** (`tailwindcss@3.4.15` in package.json) to prevent PostCSS integration compile conflicts.
* Design system follows a dark-mode first design palette.
* Focus styles must be present for interactive elements (e.g. `focus:ring-2 focus:outline-none`).
* Backdrop filters (`backdrop-blur-md`) and shadows should be applied to panels to create premium glassmorphic surfaces.
